# **App Name**: Career Compass AI

## Core Features:

- AI Chatbot Counselor: Provide career suggestions and advice through an interactive AI-powered chatbot tool.
- AI Career Quiz: Assess user interests and skills through a multiple-choice quiz to provide personalized career path recommendations.
- Resume/Skill Analyzer: Match user skills (uploaded or input) with ideal career options.
- Career Explorer: Browse popular career categories with filters and detailed information.
- Community Forum: Enable students to discuss and share guidance.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke a sense of calm and trust.
- Background color: White (#FFFFFF), providing a clean and modern backdrop that allows vibrant elements to stand out.
- Accent color: Sky blue (#87CEEB) for highlights and interactive elements, adding a youthful and energetic touch.
- Body font: 'PT Sans', a humanist sans-serif to maintain readability and a modern feel.
- Headline font: 'Space Grotesk' a geometric sans-serif with a precise, contemporary feel.
- Use modern, minimalist icons that align with the futuristic theme.
- Implement rounded cards and visual tiles for career categories to create a friendly and approachable interface.
- Use typing animations for taglines and smooth transitions for a modern, interactive feel.
